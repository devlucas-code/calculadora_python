# 🧮 Calculadora em Python

Uma calculadora simples e funcional desenvolvida em Python, com interface intuitiva e operações matemáticas básicas.

## 📋 Descrição

Este projeto implementa uma calculadora em Python que permite realizar operações matemáticas fundamentais como adição, subtração, multiplicação e divisão de forma fácil e prática.

## 🎯 Funcionalidades

- ✅ Adição (+)
- ✅ Subtração (-)
- ✅ Multiplicação (*)
- ✅ Divisão (/)
- ✅ Interface amigável
- ✅ Validação de entrada

## 🚀 Como Usar

### Pré-requisitos

- Python 3.6 ou superior

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/devlucas-code/calculadora_python.git
```

2. Navegue até o diretório do projeto:
```bash
cd calculadora_python
```

3. Execute o programa:
```bash
python main.py
```

## 💻 Exemplos de Uso

Após executar o programa, siga as instruções na tela para:
1. Inserir o primeiro número
2. Escolher a operação desejada (+, -, *, /)
3. Inserir o segundo número
4. Visualizar o resultado
